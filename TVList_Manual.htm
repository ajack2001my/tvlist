TVList v1.0.2 — User Manual

An HTML generator for keeping track of your favorite shows from EZTV.

Developed by Adrian Chiang. Public domain.

1. Overview

TVList reads a plain-text data file (tvlist.txt) and generates an HTML page (tvlist.htm) with search links to EZTV, torrent sites, TV news/profiles, streaming sites, and Google for each show you track. It also includes EZTV bookmark quick-checks, jQuery countdown timers for upcoming shows, a toggle-able table, and optional fancy sorting.

2. Quick Start

  1. Edit tvlist.txt with your shows (see Section 4 for format).
  2. Optionally edit tvlist.cfg for colors, fonts, and preferences.
  3. Run tvlist (or tvlist /s for sorted output).
  4. Open tvlist.htm in your browser.

3. Installation & Requirements

TVList is a Free Pascal-compiled command-line program for Windows (64-bit). It requires:

  • A Windows 64-bit system
  • No installation — just run tvlist.exe
  • A modern web browser (Firefox, Chrome, Edge, Opera) to view the output

4. The Data File (tvlist.txt)

Each line in tvlist.txt represents one TV show. Lines starting with # are ignored. The general format is:

[YYYYMMDD]StatusChar|ShowName^Group,

4.1 Status Characters

CharMeaningDisplay
0Air day: Sundays------
1Air day: Monday-M-----
2Air day: Tuesday--t----
3Air day: Wednesday---W---
4Air day: Thursday----T--
5Air day: Friday-----F-
6Air day: Saturday------S
XIn Development[INDEV]
YOn Break[BREAK]
ZEnded[ENDED]

If no status character is present, the show is marked as [UNKWN] (Unknown).

4.2 Countdown Dates

Add a [YYYYMMDD] prefix to display a jQuery countdown showing days until that date. Example:

[20260715]3|Show Returning Soon,/shows/12345/

4.3 Anime Flag (^)

Add ^GroupName after the show name to flag it as anime. This adds a search link to Nyaa and uses the group name for more targeted searches on LimeTorrent.

3|My Anime^SubsPlease,/shows/12345/

4.4 Truncation Flag (=)

Add = at the end of the show name to truncate the display name (adds ...). The full name is still used for search URLs.

3|A Very Long Show Name That=,/shows/12345/

4.5 Comments (@)

Add @comment text at the end of the line for a mouseover tooltip on the show name.

3|Show Name,/shows/12345/@Returns next month

4.6 Line Format: Examples

# This is a comment line and is ignored
3|Example Show,/shows/12345/example-show/
[20260704]0|Weekend Premiere^FansubGroup,/shows/67890/
X|Coming Soon=Truncated Display,/shows/11111/
Ended Show,/shows/22222/@This show has finished
6|Saturday Special,/shows/33333/

6. Command-Line Usage

tvlist [options] [basename]

OptionDescription
/sEnable FancyShow mode (sorts shows into categories: running, upcoming, other)
<basename>Use a custom base name instead of tvlist. Reads <basename>.txt and writes <basename>.htm. Example: tvlist mylist reads mylist.txt → writes mylist.htm

Examples:

C:\TVList> tvlist
C:\TVList> tvlist /s
C:\TVList> tvlist myshowlist /s

7. FancyShow Mode (/s)

When enabled (via /s or FancyShow=Y in config), TVList sorts the data file into three sections:

  1. Running Shows — Shows with status characters 0-6 (currently airing)
  2. Upcoming Shows — Shows with [YYYYMMDD] countdown dates (sorted alphabetically)
  3. Other Shows — Everything else (shows starting with #, or none of the above)

The output is written to a temporary file and then read back for HTML generation. Progress is shown on screen: . for each separating step, @ for unmatched lines, * during merge. Temp files are automatically deleted when TVList finishes.

8. Output Features (tvlist.htm)

8.1 Table Columns

AirsAir day, status label, or jQuery countdown timer

TV ProfilesNextEpisode and TVfinale links
TV NewsVariety and TVLine links
Watch OnlineYesMovies streaming link

8.2 EZTV Quick Checks

The "EZTV Quick Checks" section at the bottom lists EZTV show pages grouped by air day (Sun, Mon, Tue, Wed, Thu, Fri, Sat, Others). Click a day link to open multiple EZTV pages for that day's shows in new tabs.

Note: Firefox may block multiple pop-ups. Either change dom.block_multiple_popups to false in about:config, or allow pop-ups for the page in Chrome/Opera.

8.3 Toggle Table

The Toggle Shows button hides/shows the data table. Useful when the EZTV Quick Checks section goes off-screen.

8.4 Easter Eggs

  • Random humorous quote in the footer (200 quotes included)
  • Snowflake animation from Dec 18 to Dec 31 (configurable)
  • π symbol link to cDc website

8.5 Site Status Dropdown

Use the "Site Status" dropdown to check if EZTV, 1337x, and other torrent sites are currently accessible via isitdownrightnow.com.

8.6 Subtitles Utility

The "Subtitles Utility" dropdown provides links to translate, extract, convert, and locate subtitle files.

9. Compiling from Source

TVList is written in Free Pascal and compiled with:

fpc -Twin64 -Px86_64 -O3 -B tvlist.pas

The _Compile.cmd script handles the full build (requires FPC installed and the ww.bat environment setup).

Source code is available at: github.com/ajack2001my/tvlist

10. Known Limitations

  • Multiple pop-ups: Firefox blocks multiple window.open() calls by default. See Section 8.2 for workarounds.
  • Config file parsing: Values must have no spaces around the = sign (e.g., Key=Value, not Key = Value).
  • File naming: Custom base names cannot contain dots (a dot marks a parameter as an output filename override).
  • Data file: Maximum 512 EZTV pages tracked, 10,000 lines for sorting in FancyShow mode.
  • HTML output: No HTML encoding on show names — special characters like <, >, " in show names may render oddly or break the HTML.

11. Troubleshooting

ProblemSolution
"Error, file tvlist.txt not found"Ensure tvlist.txt exists in the same directory as tvlist.exe.
Config changes not appliedMake sure lines are not commented out with #, and there are no spaces around =.
FancyShow not workingUse /s flag or verify FancyShow=Y (not commented) in config.
Browser shows multiple pop-ups blockedConfigure your browser to allow pop-ups from this page (see Section 8.2).
Anime search not appearingEnsure show name contains ^ flag (e.g., Show Name^Group).
Countdown not workingVerify the date format is exactly [YYYYMMDD] at the start of the line.
